# Check System Spec

## Command

- `lscpu` list CPU and processor info
- `hwinfo` generic hardware information
- `lspci` PCI busses, including graphics card, network adapter
- `lsblk` list block devices (storate and partitions)
- `df -h` disk free
- `free -h` total, free, used RAM
